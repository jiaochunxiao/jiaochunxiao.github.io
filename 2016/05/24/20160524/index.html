linux常用命令学习(十) -- nl

发表于 2016-05-24 更新于 2025-04-17 分类于 linux 阅读次数： Valine：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

linux关于文件行号显示，cat / nl 命令均可以实现，但nl 可以将行号做比较多的显示设计，包括位数与是否自动补齐 0 等等的功能。

命令格式

nl [选项] [文件]

常用命令参数

参数功能
-b-b指定行号显示有两种显示结果：-b a 无论是否为空行，均显示行号(类似 cat -n); -b -t 仅对非空行号显示，空白行不标记行号
-n-n ln 行号在行号栏最左方显示；-n rn 行号在行号栏最右方显示,且不加0；-n rz 行号在行号栏最右方显示,且加0
-w行号栏占用位数
-p在逻辑定界符处不重新开始计算

示例

  • nl -b a 无论是否为空行，均显示行号
  • nl -b t 仅对非空行号显示，空白行不标记行号
  • nl -n ln 行号在行号栏最左方显示
  • nl -n rn 行号在行号栏最右方显示,且不加0
  • nl -n rz 行号在行号栏最右方显示,且加0
  • nl -b a -n rz
  • nl -b a -n rz -w 行号显示位数
